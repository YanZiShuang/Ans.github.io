Install Emacs On Gentoo

Emacs 下在Gentoo默认安装是不带X的，要通过自定义USE来进行安装这篇文章将会主要以我使用的USE进行安装。

首先创建packages.use目录mkdir -pv /etc/portage/package.use/然后创建并编辑/etc/portage/package.use/emacs文件，其内容如下： 


app-editors/emacs X xft xwidgets dbus png imagemagick jpeg
app-emacs/emacs-common-gentoo X

之后运行emerge -av emacs就可以得到你的emacs啦！Happy Hacking Emacs!
